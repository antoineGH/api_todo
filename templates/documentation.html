Product CRUD
Requires JWT Token auth as admin

GET /api/admin/products Return JSON with all products. https://flask-shop-application-api.herokuapp.com/api/admin/products

POST /api/admin/products Post new product from JSON https://flask-shop-application-api.herokuapp.com/api/admin/products

{"product_name": "product_name", "product_description": "product_description", "price": "500", "stock": "stock", "images_url": "images_url", "category_id": "category_id"}

GET /api/admin/product/id Return JSON selected product. https://flask-shop-application-api.herokuapp.com/api/admin/product/id
